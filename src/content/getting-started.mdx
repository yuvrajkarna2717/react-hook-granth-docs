## Installation

Install the package using your preferred package manager:

```bash
# npm
npm install react-hook-granth

# yarn
yarn add react-hook-granth

# pnpm
pnpm add react-hook-granth
```

## Quick Start

Import any hook and start using it immediately:

```jsx
import { useCounter, useDebounce } from 'react-hook-granth';

function MyComponent() {
  const { count, increment, decrement } = useCounter(0);
  const debouncedCount = useDebounce(count, 500);

  return (
    <div>
      <p>Count: {count}</p>
      <p>Debounced: {debouncedCount}</p>
      <button onClick={increment}>+</button>
      <button onClick={decrement}>-</button>
    </div>
  );
}
```
