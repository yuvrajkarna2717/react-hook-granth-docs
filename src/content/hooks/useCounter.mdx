# useCounter

A versatile hook for managing counter state with increment, decrement, reset, and set functionality.

## Installation

```bash
npm install react-hook-granth
```

## Basic Usage

```tsx
import { useCounter } from "react-hook-granth";

function Counter() {
  const { count, increment, decrement, reset } = useCounter(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={increment}>+</button>
      <button onClick={decrement}>-</button>
      <button onClick={reset}>Reset</button>
    </div>
  );
}
```

## API Reference

### Parameters

| Parameter      | Type     | Default | Description                      |
| -------------- | -------- | ------- | -------------------------------- |
| `initialValue` | `number` | `0`     | The initial value of the counter |

### Return Value

The hook returns an object with the following properties:

| Property    | Type                      | Description                                        |
| ----------- | ------------------------- | -------------------------------------------------- |
| `count`     | `number`                  | The current counter value                          |
| `increment` | `() => void`              | Function to increment the counter by 1             |
| `decrement` | `() => void`              | Function to decrement the counter by 1             |
| `reset`     | `() => void`              | Function to reset the counter to its initial value |
| `set`       | `(value: number) => void` | Function to set the counter to a specific value    |

## Advanced Usage

### With Custom Initial Value

```tsx
const { count, increment, decrement, reset } = useCounter(10);
```

### Setting Specific Values

```tsx
const { count, set } = useCounter(0);

// Set counter to 100
set(100);
```

### Building a Step Counter

```tsx
function StepCounter() {
  const { count, increment, decrement, reset, set } = useCounter(0);

  const incrementBy = (step: number) => {
    set(count + step);
  };

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => incrementBy(5)}>+5</button>
      <button onClick={() => incrementBy(10)}>+10</button>
      <button onClick={reset}>Reset</button>
    </div>
  );
}
```

## Use Cases

- **Form Step Navigation**: Track current step in multi-step forms
- **Pagination**: Manage current page number
- **Quantity Selectors**: Product quantity in shopping carts
- **Game Scores**: Track points or lives in games
- **Timer Counters**: Count down or up for timers

## TypeScript

The hook is fully typed and provides excellent TypeScript support:

```tsx
import { useCounter, UseCounterReturn } from "react-hook-granth";

// All return values are properly typed
const counterState: UseCounterReturn = useCounter(0);
```
